# RideLoop Frontend

Person B React SPA for a smart carpooling and ride-sharing application.

## Included Person B Scope

- React app scaffold with routing and CSS setup
- Shared navbar, footer, and protected route wrapper
- Local sign-in/sign-up with password, auth context, JWT placeholder, and customer/driver roles
- Editable user profile with profile photo URL field
- Marketplace ride cards with search and category filters
- Ride detail page with seller info and inquiry button
- Create/edit ride listing form
- Sent/received inquiries dashboard with accept and decline actions
- Star rating component, review form, and profile reviews list
- Responsive glassy UI with loading, empty, saved, and status states

## Run Locally

```bash
npm install
npm run dev
```
